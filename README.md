## Description

Sample Python keylogger application using [pynput](https://pynput.readthedocs.io/en/latest/index.html).

Example from:
https://pynput.readthedocs.io/en/latest/keyboard.html

## Requirements

- Python 3.9.2
- pip

## Setup

1. Install pynput.

```
pip install pynput
```
